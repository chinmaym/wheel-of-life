Mobile Design Notes

  • Stack-based layout: Wheel chart takes up most space, buttons below
  • Touch-friendly: Large buttons (44px minimum height) for easy tapping
  • One-column form: Questions display full-width with 1-10 buttons wrapping
  • Summary badges: Show lowest scores prominently to highlight problem areas
  • Fast interaction: Form can be completed in ~2 minutes on mobile
  • Responsive breakpoint: Switch to single column below 600px width

Screen Descriptions

Screen 1: Home/Dashboard

The main landing screen after opening the app. Displays the user's current life balance as a wheel/radar chart. The chart uses color coding to indicate satisfaction levels in each life area.

  • Prominent wheel visualization (radar chart)
  • Shows last check-in timestamp
  • Two main CTAs: "New Check-in" and "View History"
  • Optional: Show quick insights ("Your Finances need attention!")
üì± Mobile: Same layout, but wheel chart height reduced. Chart should be ~250px tall on mobile, with buttons below.

Screen 2: Check-in Form

Interactive questionnaire where users rate 8 life areas on a 1-10 scale. One category per card with emoji icon.

Life Categories: Health, Career, Finances, Relationships, Family, Personal Growth, Fun/Recreation, Spirituality

  • Question: "How satisfied are you with [Category]?" (1-10 scale)
  • Visual feedback: Selected number highlighted in purple
  • All 8 questions on one scrollable page
  • Submit button at bottom
  • Estimated completion time: 1-3 minutes
üì± Mobile: Full-width cards, number buttons wrap to next line naturally. Single column layout.

Screen 3: History & Trends

Shows past check-ins with timestamps and overall scores. Users can track progress over time.

  • List of recent check-ins (newest first)
  • Shows date, time, and overall score
  • Optional: Expand item to see category breakdown
  • Optional: Line chart showing score trends
üì± Mobile: Same list format, full-width cards. Tap to expand details inline.

Color Scheme & Branding

  • Primary Color: #667eea (Purple-blue) - for buttons, highlights, headers
  • Secondary Color: #764ba2 (Deep purple) - for gradients
  • Success/Green: #10b981 (for high scores, 8-10)
  • Warning/Yellow: #f59e0b (for medium scores, 5-7)
  • Alert/Red: #ef4444 (for low scores, 1-4)
  • Text: #333 (dark), #666 (medium), #999 (light)
  • Background: #f5f5f5, #fafafa

Typography

  • Font Family: System fonts (San Francisco, Segoe UI, Roboto) for best performance
  • Headings: 24px (main), 20px (section), 16px (card)
  • Body: 14-16px for readability
  • Small text: 12-13px for timestamps, hints
  • Weight: 600 for buttons and highlights, 400 for body text
